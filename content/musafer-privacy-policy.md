Privacy-relevant Android permissions requested by this app

CAMERA
======
Used for permissions that are associated with accessing camera or capturing images/video from the device.

This permission is required by the social networks integration in the Musafer application (Twitter, Facebook and Instagram). This allows the user of the Musafer application  to create posts that include photos taken using the camera. No user camera data is stored by the Musafer application.

LOCATION
========
Used for permissions that allow accessing the device location. The Musafer application needs access to the users device location in order to provide location-based information to the application user. The location-based information that are provided are:
- Weather information
- Islamic prayer times
- Qibla direction
- Currency converter
- Foursquare points of interest<
- Travel deals

PHONE
=====
Used for permissions that are associated telephony features. The Musafer application only requires read only access to phone state, including the phone number of the device, current cellular network information, the status of any ongoing calls, and a list of any Phone Accounts registered on the device. This permission is required by the Musafer application as enforced by the Android SDK requirements. The Musafer application does not use or store or share any phone related.

STORAGE
=======
Used for runtime permissions related to the shared external storage. This permission allows the Musafer application to download travel reports to the user device. The user is able to view downloaded reports after they are saved to the device.
