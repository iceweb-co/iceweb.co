# Musafer Privacy Policy

Privacy-relevant Android permissions requested by this app

## 1. CAMERA

Used for permissions associated with accessing camera or capturing images/video from the device.

The social networks integration in the Musafer app (Twitter, Facebook and Instagram) requires this permission. This allows the user of the Musafer app  to create posts that include photos taken using the camera. The Musafer app does not store user camera data.

## 2. LOCATION

Used for permissions that allow accessing the device location. The Musafer app needs access to the users device location to provide location-based information to the app user. The location-based information provided are:
- Weather information
- Islamic prayer times
- Qibla direction
- Currency converter
- Foursquare points of interest<
- Travel deals

## 3. PHONE

Used for telephony features permissions. The Musafer app requires read-only access to phone state, including the phone number of the device, current cellular network information, the status of any ongoing calls, and a list of any Phone Accounts registered on the device. The Android SDK requires this permission. The Musafer app does not use or store or share any phone related.

## 4. STORAGE

Used for runtime permissions related to the shared external storage. This permission allows the Musafer app to download travel reports to the user device. The user is able to view downloaded reports after the user saves them to the device.
